# draadloze communicatie proof of concept
minimale hard- en software waarmee aangetoond wordt dat duplex kan gecommuniceerd worden tussen de microcontroller en een [laptop|smartphone], gebruik makend van [Arduino IDE en Serial Bluetooth Terminal]
<br />
### configuratie
Baudrate Bluetooth 19200  
RX pin van HC08 op D8  
TX pin van HC08 op D7  

Baudrate Seriele 115200

### gebruiksaanwijzing
Stuur test via de app of seriele monitor. Deze stuurt TEST terug naar seriele monitor of app, afhankelijk waar je deze command hebt gestuurd.  
De ingebouwde Led word getoggled door test te sturen.
